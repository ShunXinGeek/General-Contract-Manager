# 通用合同管理助手 (General Contract Manager)

## 📌 项目概述
通用合同管理助手是一个独立运行的桌面级“壳”应用程序。它完全移除了硬编码的特定合同逻辑，转而被设计为一个**高度泛化、完全数据驱动**的合同内容管理、富文本编辑和多模态 AI 分析平台。

**核心理念**：只要用户将任意的合同条款数据（如 GCC 文本、SCC 文本、租赁协议等）按约定格式整理导入，本程序便能**自动识别、动态注册多级标签页、深度解析引用关系**，并完全接管条款呈现、内容检索、精准的版本同步滚动与前沿大模型对话分析等全流程功能。

---

## 🏗️ 架构与技术栈

项目基于 **Electron** 框架重构，整合了纯原生前端技术栈 (Vanilla JS + HTML + CSS)，兼顾了系统的底层交互与前端轻量级渲染：
- **无依赖独立运行**：软件内置 Node.js 与 Chromium 引擎，使用者**无需安装**任何外部环境（如 Python 或浏览器环境），即可直接双击运行。
- **系统级集成**：支持“后台托盘运行”、“全局快捷键唤醒”、“窗口强力置顶”、“硬件加速”及“自动检查更新”等原生桌面级特性。
- **本地+云端结合架构**：配置和关键数据默认安全隔离在本地设备的 LocalStorage 及 AppData 内；也可一键漫游至 Firebase 数据库（或预留了后续 COS 等流转改造的底层）。

## 🧩 各类创新功能说明

### 1. 通用桌面功能支持 (Module: main.js & ai-settings.js)
点击主界面右上方的“⚙️设置”图标，进入**通用设置**面板：
- **最小化到托盘**：关闭窗口不退出程序，守护进程后台运行。
- **全局快捷键**：工作时按下 `Ctrl+Shift+Space` (依您的设定) 秒级唤出合同助手，极大提升生产力。
- **窗口置顶**：当对比本地文档时，可固定程序悬浮在屏幕最上层。

### 2. 高阶编辑模式与本地带格式流导出 (Module: editor.js)
一键提取经过修改或添加备注（高亮🖍️、便笺📝）的长篇条款，**并打包导出至 Word (.docx) 及 PDF**。其中 Word 导出模式完美嵌入了原生多级编号与段落缩进。系统同时设置**编辑模式锁🔒**，平时防误触，开启后才可自由更正、剪裁文本。

### 3. 多模态 AI 智能防歧义分析系统 
左上侧引入独立的“管理助手”界面：
- 集成对任意大模型通道的自定义兼容（兼容 OpenAI / DeepSeek 等顶流通道 API，支持通过 Base URL 和密钥动态增减自定义模型）；
- 在 AI 会话中，彻底解决由于条款层级重叠引发的回溯死角。精准识别类似 “SCC 61B” 并直接联动右侧面板 (Reference View) 唤起原文及其译本的双线检视。

### 4. 数据导入与动态重构隔离区
无论用户导入 `GCC` 或 `SCC` 数据，系统都会智能建立并隔绝所属独立的 Tabs 作用域，完全隔离状态，防止多合同审查时的混淆。

## 📖 操作与安装说明

### 安装与运行 (使用者)
1. **获取软件**：直接获取最终的安装包 (如 `通用合同管理助手 Setup 1.0.0.exe`)。
2. **安装并打开**：双击即可完成安装。安装后在桌面或开始菜单找到快捷方式，双击即可无缝启动程序，没有任何环境配置的烦恼。

### 源代码运行与打包 (开发者)
如果您需要获取源码二次开发：
1. **安装依赖**：在此项目根目录下运行终端，执行：
   ```bash
   npm install
   ```
2. **调试运行**：
   ```bash
   npm start
   ```
3. **打包生成 EXE 可执行文件**：
   开发完毕后，可执行以下命令自动通过 `electron-builder` 在 `dist` 目录下构建输出安装程序：
   ```bash
   npm run dist
   ```

### 软件日常使用指南
1. **导入挂载数据**：
   - 启动后中心显示“请导入合同数据”。
   - 点击界面上方 **“📥 导入数据”** 图标，选择导入您备好的 `.txt` 或之前存档的 `.json` 格式的合同。
   - 导入后，页面会自动生成合同名称标签卡栏。

2. **配置 AI 与软件平台**：
   - 点击右上角的 **“⚙️ 设置”** 图标。
   - 在**“AI 助手设置”**内，添加并填入您自己的模型参数（如 DeepSeek、GLM-4 等 API Key 及节点地址）。
   - 在**“通用设置”**内个性化调整您的快捷键和托盘习惯。

3. **开始利用智能化办公**：
   - 点击左上角的 **“🤖 管理助手”** 发起提问。
   - 如果遇到重要条款，可选中该段条款内容，并点击聊天框上的 **“📋 引用条款”** 快速放入 AI 的上下文。
   - 需要沉淀知识时，划择条款直接使用右侧悬浮工具条（加粗、改色、写批注等），最后通过顶部 **“📤 导出合同文档”** 留存记录。
