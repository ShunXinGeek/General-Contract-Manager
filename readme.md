# HK Contract Manager / 通用合同管理助手 (General Contract Manager)

## 📌 项目概述
通用合同管理助手是一个全新重构的、无固定合同类型的纯粹“壳”应用程序。它完全移除了硬编码的特定合同逻辑，转而被设计为一个**高度泛化、完全数据驱动**的合同内容管理、富文本编辑和多模态 AI 分析平台。

**核心理念**：只要用户将任意的合同条款数据（如 GCC 文本、SCC 文本、租赁协议等）按约定格式整理导入，本壳程序便能**自动识别、动态注册多级标签页、深度解析引用关系**，并完全接管条款呈现、内容检索、精准的版本同步滚动与前沿大模型对话分析等全流程功能。

---

## 🏗️ 架构与代码组成

项目采用纯原生前端技术栈 (Vanilla JS + HTML + CSS) 构建，轻量高效且无需绑定重型后端服务器即可运行（AI 对话代理直接对接第三方 API）。数据持久化默认依赖浏览器本地缓存 `localStorage`，同时全面支持一键漫游至**腾讯云对象存储 (COS) 等云端平台**。 

为保证程序的长期可拓展性，系统底层代码现已全面实行**模块化重构**：

## 🧩 各类创新功能说明

### 1. 动态与强容错隔离机制 (Module: app.js & import.js)
无论用户导入的是 `GCC.txt` 抑或是 `SCC.txt`，亦或合并包含二者的文件块，系统都会智能建立并隔绝所属独立的 Tabs 作用域，并杜绝标签 ID 的任何混乱碰撞。

### 2. 高阶编辑模式与原生的 Word / PDF 流导出 (Module: editor.js)
系统增添了极为易用的文本导出操作：一键提取经办人修改过的文本片段或者加入备注后的长篇条款，**并打包导出至 Word (.docx) 及 PDF，这不仅完美保持内容纯度，其 Word 输出时更能直接享有内置的 Word 本地缩进系统和原生多级编号。**
同时内建了**严格输入防护 (编辑模式开关🔒)**；平时关闭该锁头编辑状态时，系统会硬核防止所有删除、剪切和误触粘贴，用户绝不会因为误碰键盘而损坏严谨的原始条款法源，同时依然支持标记各类高亮(`🖍️`)及便笺(`📝`)。且系统配置了“Clear All (清空重置)”快速状态回归功能。

### 3. 多模态 AI 大模型与上下文智能防歧义分析 (Module: ai-* 等模块)
左上侧引入独立的“管理助手”界面：
- 集成对任意大模型通道的自定义兼容（包含且不局限于 OpenAI / DeepSeek 等顶流通道）；
- “思考 (Thinking)”机制能够展现推理逻辑；
- 在 AI 对话反馈文本中，应用已经重写正则与跳转逻辑，彻底解决超链接标题识别时的重叠报错（如修复了“SCC SCC 61B”重复命名异常）。并精确到“SCC Clause X”或“GCC Clause X”独立绑定到对应条款进行原文精准定位回流。

### 4. 数据完全掌控，腾讯云平台流转支持 (Module: cloud-storage.js)
原先局限于 Firebase 的强绑定逻辑已经被移除，更新为完全开放的 `cloud-storage.js` 底座向通用型云生态平台靠拢，且默认提供了无缝平滑对接腾讯云 OSS (对象存储) 备份数据及迁移指南。让数据留存在国区本土、满足企业上云的审核合规标准。

## 📖 操作流程极简指北

### 第一步：轻量级唤醒启动
1. 本程序**不能直接对文件双击运行**，否则易遭遇各大浏览器极严苛的安全策略限制（阻止跨源与 Service Worker 特性拉起）。
2. 在您的目录终端直接敲入 `python -m http.server 8080` 并借由 `http://localhost:8080` 进入程序。看到居中闪耀的“请导入合同数据”画面即表示完全就绪。

### 第二步：导入挂载条款文档
1. 观察页面顶部，直接定标向右上侧代表 **“📥 导入数据”** 的图示按钮。
2. 呼出选项并在弹框内自由选择是 **“1. 导入 (.json) [恢复书签]”** 或是 **“2. 导入合同条款 (.txt) [装载全新内容]”** 以送入本地撰写完毕的新文件。
3. 一旦导入成功，页面正上方的标签层自然涌现实时标题供您查阅审读；左侧的树状目录菜单也同步刷新展示层级导航！

### 第三步：玩转大模型知识引擎
1. 找到右上角的 **⚙️ 齿轮标识 (设置)** 弹出窗口配置属于您私有的 API Key 与对接端点链接 (Endpoint)。
2. 返回并切换界面核心主 Tab 至 **“🤖 管理助手”**。在聊天栏的底部自由抛去疑问，您甚至可以划择主条款文本一键使用“**📋 引用条款**”，令分析事半功倍！并能在任意复杂讨论节点通过点击界面提供的蓝色参考锚点直接双边参考（中英文本将同步呈现于面板边侧）。

### 第四步：存疑、标记批阅并完成跨端输出
发现重要疑问之处或者需留作备案调整：
- 直接**划选文稿文本**触发悬停交互，用“彩色马克笔”和“记事本便贴”将思想火花固定在内存里。即使忘记开“编辑模式锁头”也不用慌，程序能防止你误删但不阻止安全上色的权限！
- 最后点击编辑器中相应的导出按键，携带所有智能备注信息直接提取为您本地磁盘下的 `.docx` 正宗可二次分发流转的合同原稿或是防修改不可逆形式展现的绝佳 PDF 参考件；或点击上方“☁️ 云按钮”执行上传入腾讯云端动作即可实现与他人的异地多设备书签镜像互通。
